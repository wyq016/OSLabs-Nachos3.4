## Syllabus
See W1P6(tentative)
The Labs are about:

阅读源代码，理解源代码(!!!)

• 掌握操作系统的全局设计

• 掌握操作系统各个重要模块(关键知识点)的设计与实现


在已有框架基础上完成一个教学操作系统Nachos ：对现有系统提出修改、完善、扩展的设计方案并编码

重点部分:

 线程机制

 同步机制

 文件系统

 存储管理

 通信机制与死锁

• 了解MIPS和ARM架构，探讨Nachos在ARM上的移植方案

• 掌握Nachos开发过程中需要使用的环境和工具

• Linux环境、交叉编译工具、调试工具(gdb)
• 代码辅助阅读工具(ctags、cscope、source insight)
• 版本控制工具(Git、CVS)
